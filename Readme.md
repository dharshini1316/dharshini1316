## 👩‍💻 About Me

<img align="right" src="https://media.giphy.com/media/qgQUggAC3Pfv687qPC/giphy.gif" width="280" />

Hey, I'm Dharshini — an AI engineer who genuinely enjoys building systems that are both intelligent and secure.

I work at the intersection of AI, cloud infrastructure, and security — designing systems that don't just work, but work *safely*. Whether it's a RAG pipeline with zero-trust access control or an adversarial defense lab, I care about the "what if it breaks" as much as the "how it works."

I'm drawn to problems that require thinking across layers — from model behavior to infrastructure design to real-world deployment. I like building things that have actual impact, not just demos.

Currently focused on:

- 🔐 AI security & governance systems
- 🧠 Retrieval-Augmented Generation (RAG) pipelines
- ☁️ Cloud-native AI architectures
- 🛡️ Adversarial robustness & explainability
